计算机网络基础

七月 9, 2018 发布在 Computer Networking

计算机网络分类、设备、OSI模型。

网络的分类（地理空间）：

  • LAN - local Area Network：在同一栋建筑（学校、小办公室、家庭）内，（为什么需要？）可以共享一些（打印机）设备。
  • MAN - Metropolitan Area Network：对应城市这个范围，城市内的医院、学校、图书馆组成了MAN。
  • WAN - Wide Area Network：对应国家，不同城市之间连接（大于30 miles）。

网络连接设备：

  • 网卡 - Network Interface Card, NIC：又叫网络适配器 - Network Adapter，通过唯一的MAC地址标识自己，有有线网卡和无线网卡之分。
  • 路由器 - Router：家里的路由器。
  • 交换机 - Switch：为子网提供更多ports。种类很多，有的也有中继器的功能。
  • 中继器 - Repeater：无线信号放大器。

两台计算机是如何传输数据的？

OSI model - Open System Interconnnection Reference Model：

  1. Physical 物理层：传输比特数据。所以中继器工作在物理层。
  2. Data Link 数据链路层：确保数据帧传输正确、能按正确的顺序恢复。NIC网卡
  3. Network 网络层：决定数据的路径选择和转寄，将网络表头（NH）加至数据包，以形成分组。网络表头包含了网络数据。如IP、ICMP等。所以路由工作在网络层。
  4. Transport 传输层：把传输表头（TH）加至数据以形成数据包。传输表头包含了所有使用的协议等发送信息。如TCP、UDP等。
  5. Session 会话层：负责在数据传输中设置和维护电脑网络中两台电脑之间的通信连接。
  6. Presentation 表示层：把数据转换为能与接收者的系统格式兼容并适合传输的格式。
  7. Appliction 应用层：提供为应用软件而设计的接口，如HTTP、HTTPS、FTP、TELENT、SSH、SMTP、POP3等。
